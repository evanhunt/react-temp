## 使用

```shell
# 下载
npm install

# 缓存
npm run build:dll

# 开发
npm run dev

# 发布
npm run build
```

#### `Tips`

```
npm install || yarn install

TaoBao源 npm config set registry http://registry.npm.taobao.org/

官方源 npm config set registry https://registry.npmjs.org/
```
